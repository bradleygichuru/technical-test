## Installing dependencies
in your powershell or unix base terminal type:
```console
cd backend-task
npm i
cd ..
cd frontend-task
npm i
```
## tests
```console
npm run test
```
## Start Frontend dev server
```console
cd frontend-task
npm run dev
```
## Start Backend dev server
```console
cd backend-task
npm run dev
```
